Лабораторная работа по технологиям сетевого взаимодействия.

1) Для запуска необходимо установить модули из requirements.txt;
2) Запуск api осуществляется путем ввода команды uvicorn main:app --reload;
3) После запуска необходимо перейти на http://127.0.0.1:8000;
4) Прописать в url строке браузера http://127.0.0.1:8000/docs;
5) Далее будет доступно несколько функций: добавление, вывод пользователей и удаление пользователя по user_id;
